# Java To-Do List (Console App)

## 📌 Features

* Add tasks
* View tasks
* Delete tasks
* Input validation to prevent crashes

## 🛠️ Technologies Used

* Java
* ArrayList
* Scanner

## ▶️ How to Run

```bash
javac TodoApp.java
java TodoApp
```
