# Java To-Do List (Console App)

## 📌 Features

* Add tasks
* View tasks
* Delete tasks
* Input validation (no crashes)

## 🛠️ Technologies

* Java
* ArrayList
* Scanner

## ▶️ How to Run

```bash
javac TodoApp.java
java TodoApp
```
